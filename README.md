# 我的餐廳清單
運用node.js和express進行開發，並連接Mongodb進行CRUD功能。

## 安裝環境

+ Node.js v10.15.3
+ Express ^4.17.1
+ Express-handlebars ^3.1.0
+ mongoose ^5.6.11

## 安裝步驟
1. 下載restaurant_list資料夾，並輸入以下指令下載
```
$ git clone https://github.com/KerwinJhong/S4_A9_Restaurant.git
```
2. 從終端機安裝npm套件，輸入以下指令
```
$ npm install
```
3. 執行專案
```
$ npm run dev
```
4. 輸入網址並呈現我的餐廳清單介面
```
http://localhost:3000
```
